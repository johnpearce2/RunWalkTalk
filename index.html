Getting Started

  1. Open Run Walk Talk on the leader's iPhone.
  2. Choose Lead Group, select a channel, and set the run/walk intervals.
  3. Nearby iPhones and Apple Watches should join the same channel.
  4. The leader presses Start. Everyone receives the same countdown and cues.

Common Questions

I cannot hear cues over music.
Start the run from the app, then leave your music playing. Run Walk Talk is designed to play interval cues over other audio.

My Apple Watch did not start.
Open the watch app before the leader starts, keep it near the leader phone during setup, and make sure it shows it has joined or is waiting for the leader.

Do I need internet?
Nearby group sync is designed for local setup and start. Health saving and route display use Apple services available on the device.

Health And Location

Run Walk Talk asks for Health permission only to save completed workouts. Location permission is used for route, distance, and pace features when you choose to use them.
