How it works.

No long codes, no complicated setup. For group runs, one leader starts everything from an iPhone. Nearby runners join, receive the same start, and everyone follows the same run/walk clock.

Leader chooses the interval on iPhone

For group sync, the leader needs an iPhone. They pick a channel, set run and walk times, and wait for nearby phones and watches to join.

Followers wait for the start

Follower phones and watches do not start the group themselves. They listen for the leader's iPhone and arm themselves for the shared countdown.

Everyone runs the same clock

After kickoff, each device follows the shared start time locally, keeping cues aligned while reducing battery use.

Solo runners can use it too

If you are training alone, just choose your run/walk interval and start. No group, channel, or watch sync is needed.

Made for the people at the back too.

Run Walk Talk helps leaders support the whole group, not just the runners nearby. Future versions are planned to add talk features for checking in, sharing updates, and keeping everyone included throughout the pack. Garmin watch support is also planned for a later release.

Built around the real run.

Fast setup before you move, simple cues while you run, and optional workout saving when you finish.

Privacy-minded by design.

Health access is used to save workouts only when you allow it. Location is used for route, pace, and distance features when you choose to use route tracking.

No internet required for local starts.

Nearby group sync is designed for the initial kickoff. Once devices have the shared start time, they keep running the interval clock locally.

Quick checks

Open the watch app before the leader starts, keep devices nearby for setup, and start the group from the leader iPhone.

Can I listen to music while using Run Walk Talk?

Yes. Run Walk Talk is designed so interval beeps can play over music during a run.

Does the leader need an iPhone?

Yes, for group sync. Think of the leader iPhone as the starter signal for the whole group. Watches and follower phones receive that start and then keep time themselves.

Does every runner need an Apple Watch?

No. Runners can use iPhone, Apple Watch, or both depending on what they have. Garmin watch support is planned for a future release.

Can I use it on my own?

Yes. Run Walk Talk also works as a basic solo interval timer for run/walk training.

Does nearby sync need internet?

The local group kickoff is designed to work nearby without relying on internet. Some Apple services, such as maps or Health syncing, may use device services separately.
